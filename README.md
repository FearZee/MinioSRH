# MinioSRH

This project was developed by Sascha and Erdem for the course "Alternative Database" By Prof. Dr. David Linner, SRH Berlin School of Design and Communication.

## Run Minio in Docker

```
$ docker-compose up -d
```

A docker container will be created.
